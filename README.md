# ignore

Gets .gitignore templates for you <3.

## Usage/Examples

```
Usage:
    ignore <command> [arguments]

Commands: 
    list                list all available templates
    get [spec [?...]]   get a template
    help                print help

Arguments:
  -path string
        Specify the output directory (default ".")

Examples:
    ignore list
    ignore get C Julia TeX
    ignore get VisualStudio
```
